Beschrijving potentiaalverloop bij piping met model 4a en een geohydrologisch model
=========================================================================================

In deze sectie wordt beschreven hoe het potentiaalverloop bij piping kan worden gemodelleerd met behulp van model 4a in combinatie met een geohydrologisch model. Model 4a is een analytisch model dat de stroming van water door een doorlatende laag beschrijft, terwijl een geohydrologisch model de ruimtelijke variatie in grondwaterstanden en stromingspatronen in een gebied simuleert.
